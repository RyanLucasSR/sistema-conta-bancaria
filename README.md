# 💳 Sistema de Conta Bancária em Java

Este é um projeto simples desenvolvido em Java com o objetivo de praticar os conceitos de Programação Orientada a Objetos (OOP).

---

## 🚀 Funcionalidades

O sistema permite:

- Criar uma conta bancária
- Depositar dinheiro
- Sacar dinheiro
- Transferir valores entre contas
- Render juros em conta poupança (5%)

---

## 🧠 Conceitos aplicados

Neste projeto foram utilizados os seguintes conceitos:

- Programação Orientada a Objetos (OOP)
- Encapsulamento
- Herança
- Organização de código
- Validação de dados

---

## 🏗️ Estrutura do projeto

- `ContaBancaria`  
  Classe principal que representa uma conta, contendo saldo, titular e operações básicas.

- `ContaPoupanca`  
  Classe que herda de `ContaBancaria` e adiciona a funcionalidade de rendimento de juros.

- `Main`  
  Classe responsável por testar o funcionamento do sistema.

---

## 💡 Objetivo

Este projeto foi desenvolvido como exercício prático para consolidar a base de OOP em Java, focando na organização do código e na aplicação de regras de negócio simples.

---
